# Web Chat Emulator

Веб‑эмулятор диалога из JSON/JS‑экспорта (включая формат `let dialogjson = {...}`) и HTML‑папки истории (например, `history_*.html`) с оптимизацией под большие файлы.

## Что умеет

- Загружает переписку из JSON/JS файла или из папки с HTML‑историей и строит чат.
- Парсит файл в `Web Worker`, чтобы не блокировать UI.
- Использует виртуализированный рендер сообщений (в DOM только видимый диапазон).
- Имеет поиск по сообщениям с переходом `Prev/Next` и подсветкой совпадений.
- Для HTML‑истории пытается показать медиа (фото/видео/аудио/файлы) из вложений.

## Запуск

1. Запустите статический сервер из корня репозитория:

```bash
python3 -m http.server 5173
```

2. Откройте:

```text
http://localhost:5173/web-chat-emulator/
```

3. Нажмите `Выбрать JSON` и загрузите JSON/JS, либо `Выбрать папку (HTML)` и выберите папку экспорта (где лежат `history_*.html`).

## Примечания

- При очень больших экспортах время обработки зависит от размера файла и устройства.
- Загрузка папки через `webkitdirectory` лучше всего работает в Chromium-браузерах (Chrome/Edge).
