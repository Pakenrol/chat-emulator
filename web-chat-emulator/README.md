# Web Chat Emulator

Веб‑эмулятор диалога из JSON/JS‑экспорта (включая формат `let dialogjson = {...}`) с оптимизацией под большие файлы.

## Что умеет

- Загружает переписку из файла и строит чат.
- Парсит файл в `Web Worker`, чтобы не блокировать UI.
- Использует виртуализированный рендер сообщений (в DOM только видимый диапазон).
- Имеет поиск по сообщениям с переходом `Prev/Next` и подсветкой совпадений.

## Запуск

1. Запустите статический сервер из корня репозитория:

```bash
python3 -m http.server 5173
```

2. Откройте:

```text
http://localhost:5173/web-chat-emulator/
```

3. Нажмите `Выбрать файл` и загрузите JSON.

## Примечания

- При очень больших экспортах время обработки зависит от размера файла и устройства.
